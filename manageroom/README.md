# 必要インストールモジュール
## 画面遷移モジュール
$ npx yarn add react-router-dom
$ npm install react-router-dom@5
*バージョン@5系がいいらしい*

## Firebaseモジュール
これがないとデータベース通信できない
$ npm -g install firebase
これが使えないときは https://zenn.dev/yusuke_docha/articles/7186f37c768a09 のサイトを参考にするとよい
